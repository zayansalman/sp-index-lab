# AGENTS.md

This file provides guidance to AI coding agents working in this repository.

**The canonical project guide is [CLAUDE.md](CLAUDE.md)** — architecture, commands,
key modules, code conventions, CI/CD, and current headline results all live there.
This file is intentionally a pointer so the two never drift apart.

Two things worth repeating for any agent:

1. **Methodology guarantees** (do not regress): universe selection is point-in-time on
   the dividend-unadjusted ranking panel (`src/data/universe.py`); all backtests are net
   of turnover-based costs (`src/backtest/costs.py`); the benchmark is the S&P 500
   total-return index. Strategy selection uses a hard dev/holdout split (`DEV_END`,
   `src/research/`): develop only on the dev window, log every trial, disclose the
   deflated Sharpe, and touch the holdout once via `scripts/run_holdout.py`.
2. **Numbers drift**: the current headline figures live in
   `frontend/public/data/meta.json` (`headline` block) and are regenerated by the
   daily pipeline. Never hardcode performance numbers in components or docs.
